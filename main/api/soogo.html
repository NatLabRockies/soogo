generate(n, bounds, current_sample: numpy.ndarray | None = None, iindex: tuple[int, ...] = ()) → numpy.ndarray

Generate a sample that aims to fill gaps in the search space.

This algorithm generates a sample that fills gaps in the search space. In each iteration, it generates a pool of candidates, and selects the point that is farthest from current sample points to integrate the new sample. Algorithm proposed by [1].

When no current sample points are provided, the algorithm falls back to generating random samples using the Latin Hypercube Sampling method.

Parameters:

  • n – Number of sample points to generate.

  • bounds (sequence) – List with the limits [x_min,x_max] of each direction x in the space.

  • current_sample (Optional[ndarray]) – Sample points already drawn. (default: None)

  • iindex (tuple[int, ...]) – Indices of the input space that are integer. (default: ())

Return type:

ndarray

Returns:

n-by-dim matrix with the selected points.

References

[1]

Mitchell, D. P. (1991). Spectrally optimal sampling for distribution ray tracing. Computer Graphics, 25, 157–164.

soogo.sampling.dds_sample(n, bounds, probability: float, mu, sigma_ref=1.0, iindex: tuple[int, ...] = (), seed=None) → numpy.ndarray

Generate a sample based on the Dynamically Dimensioned Search (DDS) algorithm described in [2].

This algorithm generated a sample by perturbing a subset of the coordinates of mu. The set of coordinates perturbed varies for each sample point and is determined probabilistically. When a perturbation occurs, it is guided by a normal distribution with mean zero and standard deviation sigma.

Parameters:

  • n – Number of sample points to generate.

  • bounds (sequence) – List with the limits [x_min,x_max] of each direction x in the space.

  • probability (float) – Perturbation probability.

  • mu – Point around which the sample will be generated.

  • sigma_ref – Standard deviation of the truncated normal distribution, (default: 1.0) relative to a unitary interval. Default is 1.0.

  • iindex (tuple[int, ...]) – Indices of the input space that are integer. (default: ())

  • seed – Seed to initialize the random number generator. (default: None)

Return type:

ndarray

Returns:

n-by-dim matrix with the sampled points.

References

[2]

Tolson, B. A., and C. A. Shoemaker (2007), Dynamically dimensioned search algorithm for computationally efficient watershed model calibration, Water Resour. Res., 43, W01413, https://doi.org/10.1029/2005WR004723.
